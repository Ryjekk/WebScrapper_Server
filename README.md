| Server Automaticly Scrape data every hour

### Stack: 

- Node JS 14.x.x
- Express 4.17.x
- Node-corn 2.0.x
- Puppeteer 5.4.x

## Setting up your dev environment

To install locally follow the directions in the MongoDB documentation.

`npm install`

To run server

`npm start`
